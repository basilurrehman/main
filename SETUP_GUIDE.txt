# Standalone FlareSolverr - Summary

## What You Get

I've created a **standalone Python implementation** of FlareSolverr that:

✅ **Requires NO server** - No localhost, no ports, no sessions management  
✅ **Single Python file** - `flaresolverr_standalone.py` contains everything  
✅ **Easy to use** - Read URLs from file, get results in variables  
✅ **Cloudflare bypass** - Automatically detects and solves challenges  
✅ **Data storage** - Results stay in memory or save to JSON  
✅ **Direct Python** - Import and use in your code

## Quick Setup (3 Steps)

### Step 1: Install Dependencies
```bash
pip install -r requirements_standalone.txt
```

Or manually:
```bash
pip install selenium==4.39.0 undetected-chromedriver==3.5.4 certifi==2025.11.12
```

### Step 2: Install Chrome/Chromium
- **Ubuntu/Debian:** `sudo apt-get install chromium-browser`
- **macOS:** `brew install chromium`
- **Windows:** Download from https://www.google.com/chrome/

### Step 3: Create URLs file (urls.txt)
```
https://example.com
https://httpbin.org/html
https://another-site.com
```

## Usage (Pick Your Method)

### Method 1: Run the Script Directly
```bash
python3 flaresolverr_standalone.py
```
This reads `urls.txt` and saves results to `results.json`

### Method 2: Use in Python Code
```python
from flaresolverr_standalone import StandaloneFlare

flare = StandaloneFlare(headless=True)
result = flare.fetch_url('https://example.com')

print(result['html'])      # Page content
print(result['status'])    # HTTP status
print(result['cookies'])   # Extracted cookies
print(result['error'])     # Any errors

flare.close()
```

### Method 3: Batch Processing
```python
from flaresolverr_standalone import StandaloneFlare

flare = StandaloneFlare(headless=True, disable_media=True)
results = flare.fetch_from_file('urls.txt')
flare.save_results('output.json')

for result in results:
    print(f"{result['url']}: {result['status']}")

flare.close()
```

## Result Structure

Each fetch returns a dictionary:

```python
{
    'url': 'https://example.com',           # The URL fetched
    'status': 200,                          # HTTP status code
    'html': '<html>...</html>',             # Full page HTML
    'cookies': [                            # List of cookies
        {
            'name': 'sessionid',
            'value': 'abc123...',
            'domain': '.example.com',
            'path': '/',
            'expires': 1735689600,
            'httpOnly': True,
            'secure': True,
            'sameSite': 'Lax'
        }
    ],
    'user_agent': 'Mozilla/5.0...',         # Browser user agent used
    'error': None,                          # None if successful, else error message
    'timestamp': 1672531200.123             # When it was fetched
}
```

## Common Use Cases

### Web Scraping
```python
from flaresolverr_standalone import StandaloneFlare
from bs4 import BeautifulSoup

flare = StandaloneFlare(headless=True, disable_media=True)
result = flare.fetch_url('https://target-site.com')

if result['error'] is None:
    soup = BeautifulSoup(result['html'], 'html.parser')
    # Extract data...

flare.close()
```

### Extract Cookies for Authentication
```python
flare = StandaloneFlare()
result = flare.fetch_url('https://site-with-protection.com')

cookies = {c['name']: c['value'] for c in result['cookies']}
# Use these cookies in subsequent requests
```

### Batch Processing with Progress
```python
flare = StandaloneFlare()
results = flare.fetch_from_file('urls.txt')

success = sum(1 for r in results if r['error'] is None)
print(f"Success: {success}/{len(results)}")
```

### Save HTML Files
```python
flare = StandaloneFlare()
results = flare.fetch_from_file('urls.txt')

for i, result in enumerate(results):
    if result['error'] is None:
        with open(f'page_{i}.html', 'w') as f:
            f.write(result['html'])

flare.close()
```

## Troubleshooting

### "Chrome not found"
Install Chrome/Chromium:
- Ubuntu/Debian: `sudo apt-get install chromium-browser`
- macOS: `brew install chromium`
- Windows: Download from Google Chrome website

### "Connection refused" or "timeout"
Increase timeout:
```python
flare = StandaloneFlare(timeout=180)  # 3 minutes instead of 60 seconds
```

### "Permission denied" on script
Make it executable:
```bash
chmod +x flaresolverr_standalone.py
```

### Memory usage too high
Disable media:
```python
flare = StandaloneFlare(disable_media=True)  # No images/CSS = faster
```

## Performance Tips

1. **Headless mode is MUCH faster:**
   ```python
   flare = StandaloneFlare(headless=True)  # Default - no visual
   ```

2. **Disable media blocks resources:**
   ```python
   flare = StandaloneFlare(disable_media=True)
   ```

3. **Batch process files:**
   ```python
   results = flare.fetch_from_file('urls.txt')  # Auto-batching with delays
   ```

4. **Close browser when done:**
   ```python
   flare.close()  # Frees up memory
   ```

## Key Differences from Server Version

| Aspect | Server | Standalone |

| Sessions | Persistent | Per-request |
| Integration | Via HTTP calls | Direct import |
| Use case | Shared service | Scripting/automation |
