  $ sudo apt-get install chromium-browser

  $ brew install chromium

STEP 3: Create a file with URLs (urls.txt)
────────────────────────────────────────────────────────────────────────────
$ cat > urls.txt << 'EOF'
https://example.com

https://httpbin.org/delay/1
EOF


STEP 4: Run the script
────────────────────────────────────────────────────────────────────────────
$ python3 flaresolverr_standalone.py

This will:
  1. Open Chrome browser
  2. Fetch each URL
  3. Bypass any Cloudflare challenges
  4. Save results to results.json
  5. Close browser


STEP 5: Check the results
────────────────────────────────────────────────────────────────────────────
$ cat results.json

💻 OR USE IN YOUR PYTHON CODE
═══════════════════════════════════════════════════════════════════════════

from flaresolverr_standalone import StandaloneFlare

# Create instance
flare = StandaloneFlare(headless=True)

# Fetch a URL
result = flare.fetch_url('https://example.com')

# Access the data
print(f"Status: {result['status']}")
print(f"HTML length: {len(result['html'])}")
print(f"Cookies: {result['cookies']}")
print(f"Error: {result['error']}")

# Clean up
flare.close()
